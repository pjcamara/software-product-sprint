Some Cool Projects

  • Trie Prediction - Text Prediction in C Language

    Developed a fully functional text prediction algorithm determined by contextual co-occurence from a selected corpus using tries for efficient lookup and insertion of strings; yielding a runtime of O(k) where k is the length of an arbitrary string.

  • SimpleC Compiler - Compiler for a simplified version of C in C Language

    Developed a compiler for a simplified version of C called 'SimpleC'. The compiler currently supports arithmetic, branching, looping, parsing, and arrays.
